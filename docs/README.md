### 1. Архитектура

- **Kernel Space**:
    
    /dev/telegram/ctrl (Minor 0): Управляющий интерфейс для взаимодействия с пользовательским сервером. Через него сервер считывает запросы клиентов (read) и отправляет сформированные ответы обратно в ядро (write). 
  
    /dev/telegram/chat_1..3 (Minor 1–3): Пользовательские интерфейсы чатов. Каждое устройство соответствует отдельному независимому каналу связи.
- **User Space**: Сервер на C++20. Хранит историю (последние 10 сообщений) в `std::deque`.
- **Протокол**: Обмен через структуру `telegram_message`.

### 2. Инструкция

```Bash
make                                         # Сборка модуля и сервера
sudo insmod build/kernel/telegram_fs.ko      # Загрузка драйвера
sudo ./build/telegram_server                 # Запуск эмулятора
```
